# Remove-AzureRmSiteRecoveryVault

## SYNOPSIS

## SYNTAX

```
Remove-AzureRmSiteRecoveryVault -Vault <ASRVault> [<CommonParameters>]
```

## DESCRIPTION
The **Remove-AzureRmSiteRecoveryVault** cmdlet deletes an Azure Site Recovery vault.
