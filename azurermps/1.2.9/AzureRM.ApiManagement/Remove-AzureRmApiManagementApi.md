# Remove-AzureRmApiManagementApi

## SYNOPSIS
Remove existing API.

## SYNTAX

```
Remove-AzureRmApiManagementApi -Context <PsApiManagementContext> -ApiId <String> [-Force] [-PassThru]
```

## DESCRIPTION
Remove existing API.

## EXAMPLES

### --------------------------  Example 1  --------------------------
@{paragraph=PS C:\\\>}

```
Remove-AzureRmApiManagementApi -Context $apimContext -ApiId 0123456789
```

Remove existing API.
