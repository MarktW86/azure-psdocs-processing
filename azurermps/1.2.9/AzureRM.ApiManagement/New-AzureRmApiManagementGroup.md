# New-AzureRmApiManagementGroup

## SYNOPSIS
Creates new group.

## SYNTAX

```
New-AzureRmApiManagementGroup -Context <PsApiManagementContext> [-GroupId <String>] -Name <String>
 [-Description <String>]
```

## DESCRIPTION
Creates new group.

## EXAMPLES

### --------------------------  Example 1  --------------------------
@{paragraph=PS C:\\\>}

```
New-AzureRmApiManagementGroup -Context $apimContext -Name 'Best devs'
```

Create new group.
