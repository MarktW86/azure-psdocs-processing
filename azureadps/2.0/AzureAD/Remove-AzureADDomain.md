# Remove-AzureADDomain

## SYNOPSIS
Removes a domain.

## SYNTAX

```
Remove-AzureADDomain -Name <String> [<CommonParameters>]
```

## DESCRIPTION
The **Remove-AzureADDomain** cmdlet removes a domain from Azure Active Directory (AD).

## EXAMPLES

## PARAMETERS
